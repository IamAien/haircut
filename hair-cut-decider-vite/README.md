# Hair Cut Decider

Cute animated React + Vite page for deciding whether to cut your hair.

## Run locally

Requirements: Node.js 20.19+ or 22.12+

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

### Option A: GitHub + Vercel
1. Create a new GitHub repository.
2. Upload all files from this folder to the repo.
3. In Vercel, create a new project and import that GitHub repository.
4. Keep the default build settings if Vercel detects Vite automatically.
5. Click Deploy.

### Option B: Vercel CLI
```bash
npm install -g vercel
vercel --prod
```
